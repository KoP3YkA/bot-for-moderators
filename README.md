# Bot for moderators

## Statring

---

1) Create `.env` file and paste this to it
```dotenv
BOT_TOKEN=ur_token
MYSQL_HOST=mysql
MYSQL_USER=usr
MYSQL_DATABASE=bot
MYSQL_PORT=3306
MYSQL_PASSWORD=root
```
And change `BOT_TOKEN` to ur VK token

2) Run `docker-compose up --build` (if u already have Docker desktop)
